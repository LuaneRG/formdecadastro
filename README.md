# formdecadastro

## Descrição:

Neste projeto foi desenvolvido um formulário de cadastro com validador em javascript. O projeto implementa responsividade, validações com expressões regulares (Regex) e manipulação dinâmica do DOM para exibição de erros. As tecnologias utilizadas foram `HTML5`, `CSS3` e `JavaScript (ES6+)`."

## Como usar:

O formulário processa as entradas através de um validador customizado em JavaScript. As regras são: todos os campos são obrigatórios, o nome deve conter pelo menos 2 caracteres, a senha no mínimo 4, e o e-mail deve seguir um formato válido verificado por Regex.
